# 🤖 Smart AI Assistant (LangGraph Project)

## 🚀 Overview
This project is a smart AI assistant built using LangGraph and Groq LLM.

It can:
- 💬 Chat with users
- ➗ Solve math problems
- 🧠 Remember conversation (memory)
- 🔀 Make intelligent decisions (routing)

---

## 🧠 Features

- AI-powered routing (decides math vs chat)
- Calculator tool
- Conversation memory
- Secure API handling using `.env`

---

## ⚙️ Tech Stack

- Python
- LangGraph
- LangChain
- Groq LLM
